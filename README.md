# ebpf-loop-verification-case-study
Reproduzierbare Experimente zur Analyse von Schleifen in eBPF. Ziel ist die Untersuchung der Frage, ob und wie der eBPF-Verifier zwischen terminierenden, pseudo-unendlichen und nicht terminierenden Schleifen unterscheidet. Die Ergebnisse dienen als praktische Ergänzung zur Diskussion sicherer Kernelerweiterungen und ihrer Verifikationsverfahren.
